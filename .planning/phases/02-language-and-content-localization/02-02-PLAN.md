<objective>
Create an IP geolocation locale detection service and hook it into the login flow so that first-time users get their language auto-detected.

Purpose: Fulfills LANG-02 (automatic IP-based language detection on first visit). When a user with no locale set logs in, the server calls ip-api.com to resolve their country, maps BR to pt-BR (all others default to en-US), and persists the result to user.locale. This runs fire-and-forget so it never delays authentication.

Output: LocaleDetectionService with unit tests, updated AuthService.login() with geo-IP hook, updated AuthController with client IP extraction.
</objective>
